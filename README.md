# Projeto-Blog
 Projeto que pretende simular um blog/rede-social. Com foco em utilizar o Javascript como tecnologia principal. </br>
 A ideia é demonstrar conhecimentos e habilidades com a linguagem, sem deixar de lado a semântica HTML e um bom estilismo CSS, </br>
 nos scripts .js usei multiplos recursos oferecidos pela linguagem, além do consumo da API JsonPlaceHolder para geração de mais de 90% do conteúdo
 o qual  anexei em tags HTML através do próprio Javascript, parte pelos 'document.create' e 'appendChild' parte pela injeção de código estruturado
